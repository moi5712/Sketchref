# Sketchref

Minimalist drawing practice app for building local reference boards, running timed studies, and saving practice overlays in the browser.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the app:

   ```bash
   npm run dev
   ```
